Què és Cloudy ?

Cloudy és una distribució destinada als usuaris de Xarxes Comunitàries per fomentar la transició i l'adopció de l'ecosistema del cloud
  • Basada en Debian GNU/Linux 8 Jessie
    •
  • Per a diferents arquitectures (x86, amd64, arm...)
    •
  • Per a dispositius GRANS i petits
    •

Què és Docker ?

Docker és una eina que ens automatitza el desplegament d'aplicacions dintre de contenidors.
  • Facilitat la instal·lació i configuració de nous serveis.
    •
  • Proporciona una capa d'abstracció addicional.
    •
  • Existència de gran quantitat de contenedors 'pre-creats' "100,000+ free apps, public and private registries"
    •

Com començar ?

<web_interface> System/Updates -> Update in cDistro

Fer un install a "Docker.io" -> Installed

That's all, folks!

Com es publiquen els serveis creats amb Docker ?
  • Cloudy publica els serveis utilitzant un protocol "gossip" (xafarderies).
  • Aquesta característica ens ajuda a publicitar i cercar serveis a altres servidors dintre la xarxa.
  • ... i per tant necessitàvem que els containers docker facin servir el mateix sistema.
